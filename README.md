## 프로젝트 목적
* 'VR'과 '게임'요소를 결합해 흥미를 유발하여 환경 정보 효과적 전달
* 환경문제 해결에 대한 경각심 고취
* 환경 관련 정보전달성 게임의 필요성 강조 및 방향성 제시

* 인트로
  게임을 시작하면, 3D 애니메이션 형태의 인트로 영상이 PLAY 됨. 오염된 지구로 인해 부모님과 이별하게 된 플레이어가 부모님과 재회하기 위해 퀘스트를 수행해야한다는 전반적인 스토리 라인을 보여줌.
